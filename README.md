# CDCSM-KPNEUMONIAE

This repository contains the code and pre-trained models for antimicrobial resistance (AMR) prediction and clinical decision support in *Klebsiella pneumoniae*. It is part of the manuscript submitted to the *Journal of Infection and Public Health*.

---

# Cloning the Repository and Installation

To clone this repository to your local machine, open the terminal/command prompt and run:

```bash
git clone https://github.com/amessaoudi/cdsm-kpneumoniae.git
cd cdsm-kpneumoniae
pip3 install -r requirements.txt
