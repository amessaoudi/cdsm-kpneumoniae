# CDCSM-KPNEUMONIAE

This repository contains the code and pre-trained models for antimicrobial resistance (AMR) prediction and clinical decision support in *Klebsiella pneumoniae*. It is part of the manuscript submitted to the *Journal of Infection and Public Health*.

---

# Web Application (Streamlit Demo)

A fully interactive implementation of the Clinical Decision Support System (CDSM) is available online, demonstrating real-time antimicrobial resistance prediction and empirically ranked antibiotic recommendations.
